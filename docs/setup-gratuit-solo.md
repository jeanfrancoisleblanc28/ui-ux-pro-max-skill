# Setup IA gratuit — 1 personne, sans Azure, sans abonnement

Objectif : être efficace sur toutes les phases du travail (veille → réflexion →
production → diffusion) **sans payer un seul euro de plus**. Pas d'Azure, pas de
licence Microsoft 365 Copilot, pas de carte bancaire.

---

## La stack gratuite

| Phase | Outil gratuit | Lien | À quoi ça sert |
|---|---|---|---|
| **Assistant IA principal** | Copilot Chat | https://copilot.microsoft.com | Rédaction, analyse, résumés, images, code — gratuit avec un compte Microsoft |
| **Recherche / veille sourcée** | Perplexity | https://www.perplexity.ai | Réponses avec sources citées |
| **IA sur VOS documents** | NotebookLM | https://notebooklm.google.com | Poser des questions à vos propres PDF/notes (mini-RAG gratuit) |
| **Office (Word/Excel/PPT)** | Office sur le web | https://www.office.com | Versions en ligne gratuites avec un compte Microsoft |
| **Présentations IA** | Gamma | https://gamma.app | Génère des slides propres en quelques secondes |
| **Automatisation** | Power Automate (plan gratuit) / Zapier Free | https://make.powerautomate.com · https://zapier.com | Petits workflows automatiques |
| **Code (optionnel)** | GitHub Copilot Free + VS Code | https://github.com/features/copilot · https://code.visualstudio.com | Offre gratuite limitée mais réelle |
| **Stockage** | OneDrive (5 Go) / Google Drive (15 Go) | — | Inclus avec le compte |

## À comprendre avant de commencer

- **Copilot Chat gratuit ≠ Microsoft 365 Copilot payant (~30 $/mois).**
  La version gratuite ne s'intègre **pas** dans vos fichiers Word/Excel du bureau,
  mais elle fait l'essentiel via le navigateur : rédiger, résumer, analyser,
  coder, créer des images.
- **Le geste manuel** : vous copiez-collez entre Copilot Chat et vos documents.
  Petit effort, mais ~360 $/an économisés.
- **Aucun service Azure** n'est nécessaire pour cet usage solo.

---

## La routine d'usage (4 outils suffisent)

1. **Copilot Chat = votre QG.** Brainstorming, rédaction, analyse, génération
   d'images. Commencez toujours là.
2. **NotebookLM** quand la réponse doit venir de VOS documents (contrats, notes,
   rapports). Vous uploadez les fichiers, puis vous posez vos questions.
3. **Gamma** dès qu'il faut une présentation propre rapidement.
4. **Office Web** pour finaliser et stocker proprement.

Ces 4 outils couvrent toutes les phases : veille → réflexion → production →
diffusion.

---

## Bonnes pratiques pour aller vite

- **Gardez un prompt de contexte** réutilisable (qui vous êtes, votre secteur,
  votre ton) à coller en début de conversation Copilot.
- **Un dossier de référence** dans NotebookLM avec vos documents clés : il devient
  votre « second cerveau » interrogeable.
- **Vérifiez les sources** : Copilot et Perplexity peuvent se tromper — recoupez
  les faits importants.

---

## Et si un jour vous voulez payer (seulement si ça vaut le coup)

Le seul abonnement à envisager plus tard serait **Microsoft 365 Copilot** (~30 $/mois)
si l'intégration directe dans Excel/Word vous fait gagner beaucoup de temps au
quotidien. Tant que le copier-coller depuis Copilot Chat vous suffit, **restez sur
le gratuit.**
